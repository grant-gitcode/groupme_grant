How to set up the extension

To set up the extension, you must manually edit the popup.js file. In this file, there will be a variable called token at the top of the file. It should initially contain the value "?token=". To this, you must append your GroupMe token, so that the variable looks like "?token=mySuperLongToken". That's it!

How to use the extension

This extension allows the user to easily share links on GroupMe. Press Alt+Shift+G when on a webpage which you wish to share. In the pop up window, type the name of the group to which you wish to share the link, and enter an optional message to share with the link. Hit enter when you're ready, and the link will be shared! It's that simple!
